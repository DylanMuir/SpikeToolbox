Spike Toolbox

STPlot2DRaster
Make a raster plot of the spike train, differentiating between rows

Syntax
STPlot2DRaster(stTrain)

Description
stTrain is a mapped spike train. A raster plot will be created in the current axes (or a new figure created) showing the spike train.
