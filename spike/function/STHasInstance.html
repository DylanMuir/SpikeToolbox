Spike Toolbox

STHasInstance
Test whether a spike train contains a instance

Syntax
[bHasInstance] = STHasInstance(stTrain)

Description
stTrain is a valid spike train. bHasInstance will be true if stTrain contains a instance, and false otherwise.
