Spike Toolbox

STCreateFromVector
Package a vector of spike data as a spike train instance

Syntax
[stInstTrain] = STCreateFromVector(vSpikes) [stInstTrain] = STCreateFromVector(vSpikes, bUseISI)

Description
vSpikes should be a vector of spike times, either in absolute time stamp format, or as a set of ISIs. In either case, the times should be in seconds. These spike times will be wrapped nicely into a spike train instance.

STCreateFromVector will attempt to determine whether vSpikes is in absolute or ISI format. The user can specify an optional argument bUseISI to indicate to STCreateFromVector which format it should use.
