Spike Toolbox

STDescribe
Print a description of a spike train toolbox variable

Syntax
STDescribe(var)

Description
STDescribe will print as much information as is available about var. STDescribe calls other utility functions as necessary, when it identifies the type of var.

See Also
STAddrDescribe, STAddrSpecDescribe, STOptionsDescribe, STTrainDescribe
