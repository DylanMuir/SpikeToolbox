Spike Toolbox

STPlotDef
Make a plot of a spike train definition

Syntax
<[hFigure]> = STPlotDef(stTrain)

Description
stTrain is a spike train with a definition. STPlotDef will make a descriptive plot of the spike train definition, showing the important features.

The optional return argument hFigure will return the handle of the new figure created.
