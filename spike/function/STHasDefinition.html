Spike Toolbox

STHasDefinition
Test whether a spike train contains a definition

Syntax
[bHasDefinition] = STHasDefinition(stTrain)

Description
stTrain is a valid spike train. bHasDefinition will be true if stTrain contains a definition, and false otherwise.
