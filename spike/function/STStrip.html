Spike Toolbox

STStrip
Strip specified spike train levels from spike trains

Syntax
[stStrippedTrain] = STStrip(stTrain, strLevel, strLevel, ...) [stCellStrippedTrain] = STStrip(stCellTrain, strLevel, strLevel, ...)

Description
stTrain is a Spike Toolbox spike train object. Spike train levels listed in the function arguments will be stripped from stTrain. If all levels are stripped from a train, an empty matrix will be returned.

STStrip can accept a cell array of spike trains as its first argument. In this case, each spike train will be stripped and returned in the corresponding element of stCellStrippedTrain.
