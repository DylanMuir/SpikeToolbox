Spike Toolbox

STSieve
Filter a spike train by inter-spike interval

Syntax
[stFiltTrain, stRejectTrain] = STSieve(stTrain, fMinISI, fMaxISI)

Description
STSieve will only pass spikes with ISIs falling between fMinISI and fMaxISI (inclusive, in seconds). If either argument is supplied as an empty matrix, that argument will default to zero and infinity respectively. stTrain should be either an instantiated or mapped spike train.

stFiltTrain will be a new spike train containing all spikes matching the ISI criteria. stRejectTrain will be a new spike train containing the non-matching spikes.

Note: When an ISI falls outside the sieve criteria, both spikes that caused the failing ISI will be removed.
