Spike Toolbox

STToolboxDefaults
Creates a toolbox options structre containing the "factory default" options for the toolbox

Syntax
[stOptions] = STToolboxDefaults

Description
stOptions will be a Spike Toolbox options structure containing the default options for the toolbox. This is useful if you somehow corrupt your toolbox options.

Managing toolbox options, STOptions, STOptionSave
