Spike Toolbox

STStimServer
Send a mapped spike train to the spike server

Syntax
[stMonTrain] = STStimServer(stTrain)

Description
stTrain is a spike train mapped to a neuron/synapse address, as created by STMap. STStimServer will repeatedly stimulate with the supplied train until interrupted.
