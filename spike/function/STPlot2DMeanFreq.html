STPlot2DMeanFreq
Make a 2D plot of the mean frequency over a time interval of each pixel in a 2D array

Syntax
STPlot2DMeanFreq(stTrain) STPlot2DMeanFreq(stTrain, bPlot) STPlot2DMeanFreq(stTrain, tBin_0, tBin_f) STPlot2DMeanFreq(stTrain, bPlot, tBin_0, tBin_f)

Description
stTrain is a mapped spike train. If bPlot = 1 an imagesc type of plot will be created in the current axes (or a new figure created) showing the mean frequency of each pixel. If bPlot = 0, the figure is not created and the function will return the mean frequency of each pixel. If not specified the plot is created.
