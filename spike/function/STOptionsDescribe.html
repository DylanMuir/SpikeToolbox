Spike Toolbox

STOptionsDescribe
Display information about an options structure

Syntax
STOptionsDescribe(stOptions) STOptionsDescribe

Description
This function will print as much info as possible about the toolbox options structure stOptions. For help on obtaining a valid options structure, see STOptions. If no options structure is provided to STOptionsStructDescribe, the current toolbox options will be described.

Managing toolbox options, STOptions, STDescribe
