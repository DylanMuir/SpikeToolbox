Spike Toolbox

STIsValidOptionsStruct
Tests for a valid Spike Toolbox options structure

Syntax
[bValid] = STIsValidOptionsStruct(stOptions)

Description
stOptions is a matlab variable. If stOptions is a valid options structure, bValid will be true. Otherwise bValid will be false.
