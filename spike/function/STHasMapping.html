Spike Toolbox

STHasMapping
Test whether a spike train contains a mapping

Syntax
[bHasMapping] = STHasMapping(stTrain)

Description
stTrain is a valid spike train. bHasMapping will be true if stTrain contains a mapping, and false otherwise.
