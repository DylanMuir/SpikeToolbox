Spike Toolbox

STNormalise
Shift a spike train to time zero and fix its duration

Syntax
[stNormTrain] = STNormalise(stTrain)

Description
This function will shift the first spike in a spike train to time zero, and correct the duration fields of the spike train object to reflect the true duration of the train.
