Getting support
If you are accessing the toolbox via SVN, then you should run svn status regularly to check for bug fixes and updates. You can optionally be added to a notification list to receive an email whenever toolbox functions are modified, whether you use SVN or not.

If you do not use SVN, check the toolbox web page for updates.

To make a bug report or ask for help, email the toolbox maintainers above. The toolbox is not officially supported, but we're helpful and friendly (albeit busy) people, and we'll endeavour to help you if we can. Bug reports should always be submitted to whoever is listed as the primary toolbox contact above. Please send as much detail as you can, and preferably a snippet of MATLAB code and / or a .mat file to help us reproduce the bug.

Obtaining the toolbox
The toolbox is preferably distributed using SVN over the internet. Check the toolbox web page for information on how to request acces this way, or to download a (probably out-of-date) static copy.
